# Latex

[Greek-Symbol-Syntax](https://jblevins.org/log/greek)

To create separate equation

$$ V = mg $$

```latex
$$ V = mg $$
```

To tag a number

```latex
\tag
```

To create intext reference

```latex
\ref{equationname}
\label{equationname}
```

To begin and end equation

```latex
\begin{equation}
\end{equation}
or
\begin{equation*}
\end{equation*}
```

To add an intext formula
```latex
$equation$
```
