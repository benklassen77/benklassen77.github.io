3D Systems of Forces

Fundamental Vectors

3D Force System = a situation in which there is triaxial loading. Forces are expressed as vectors, which are composed of smaller vectors.

F⃗=Fx⃗+Fy⃗+Fz⃗=Fx⋅i^+Fy⋅j^+Fz⋅k^\vec F = \vec{F_x} + \vec{F_y} + \vec{F_z} = F_x\cdot\hat{i} + F_y\cdot\hat{j} + F_z\cdot\hat{k} F=Fx​​+Fy​​+Fz​​=Fx​⋅i^+Fy​⋅j^​+Fz​⋅k^

F⃗=F⋅u^\vec{F} = F\cdot\hat{u} F=F⋅u^

Coordinate Direction Angles = the angles between forces along specified axes and the overall force vector. α\alphaα represents the angle between Fx⃗\vec{F_x}Fx​​ and F⃗\vec FF, β\betaβ represents the angle between Fy⃗\vec{F_y}Fy​​ and F⃗\vec FF, and γ\gammaγ represents the angle between Fz⃗\vec{F_z}Fz​​ and F⃗\vec FF.

cosα=AxA, cosβ=AyA, cosγ=AzAcos\alpha = \frac{A_x}{A},\ cos\beta = \frac{A_y}{A},\ cos\gamma = \frac{A_z}{A} cosα=AAx​​, cosβ=AAy​​, cosγ=AAz​​

Position Vector = vector with two points in reference to each other rather than the origin. Algebraically, the position vector has X=(Xb−Xa)X = (X_b-X_a)X=(Xb​−Xa​) etc.

r⃗AB=(Xb−Xa)i^+(Yb−Ya)j^+(Zb−Za)k^\vec r_{AB} = (X_b - X_a)\hat{i} + (Y_b - Y_a)\hat{j} + (Z_b - Z_a)\hat{k} rAB​=(Xb​−Xa​)i^+(Yb​−Ya​)j^​+(Zb​−Za​)k^

Magnitude = numerical quantity of the force vector, found using the magnitudes of the forces along each respective axis.

How to find 3D moments

In order to find the moments, take the cross product between position vector and the force vector r⃗×F⃗\vec{r}\times\vec{F}r×F. This can be determined using the below.

Right Hand Rule = with the thumb curled in the direction of the specified axis, the fingers curl in the direction of positive moment along that axis.

Normal Procedure

  1. Check for points where slope or deflection is known.
    • Deflection is typically zero wherever there is a support
    • Slope has a negative/positive relationship where cuts meet.
  2. Solve for reactions at supports
    • Includes both moments and loads at supports.
  3. Take a cut any distance x between an end of the beam and a reaction and find Mo equation.
    • Every change in loading requires another cut
  4. Integrate equations according to the following equation.
  5. Plug in knowns and solve.
